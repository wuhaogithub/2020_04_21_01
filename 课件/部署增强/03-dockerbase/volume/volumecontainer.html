部署增强

数据卷容器

1.创建一个数据卷容器

命令格式：

docker create -v 容器数据卷目录 --name 容器名字 镜像名称 [命令(可选)]

执行效果

docker create -v /home/python/itcast --name vc-ubuntu ubuntu

2.创建两个容器，同时挂载数据卷容器

命令格式：

docker run --volumes-from 数据卷容器id/name -tid --name 容器名字 镜像名称 [命令(可选)]

执行效果：

创建vc-test1容器

docker run --volumes-from vc-ubuntu -dit --name vc-ubuntu-test1  ubuntu

创建vc-test2容器

docker run --volumes-from vc-ubuntu -dit --name vc-ubuntu-test2  ubuntu

3.确认卷容器共享

进入vc-test1，操作数据卷容器

python@ubuntu:~$ docker exec -it vc-ubuntu-test1 /bin/bash
root@c64d5171e94b:/# cd /home/python/itcast/
root@c64d5171e94b:/home/python/itcast# echo 'itcast' > test1.txt
root@c64d5171e94b:/home/python/itcast# cat test1.txt 
itcast
root@c64d5171e94b:/home/python/itcast# exit
exit

进入vc-test2，确认数据卷

python@ubuntu:~$ docker exec -it vc-ubuntu-test2 /bin/bash
root@df67044f37b4:/# cd /home/python/itcast/
root@df67044f37b4:/home/python/itcast# ls
test1.txt
root@df67044f37b4:/home/python/itcast# cat test1.txt 
itcast
root@df67044f37b4:/home/python/itcast# echo 'itheima' > test2.txt
root@df67044f37b4:/home/python/itcast# cat test2.txt 
itheima
root@df67044f37b4:/home/python/itcast# exit
exit

回到vc-test1进行验证

python@ubuntu:~$ docker exec -it vc-ubuntu-test1 /bin/bash
root@c64d5171e94b:/# cd /home/python/itcast/
root@c64d5171e94b:/home/python/itcast# ls
test1.txt  test2.txt
root@c64d5171e94b:/home/python/itcast#

结果证明：

容器间可以共享数据卷你容器，不过数据是保存在数据卷内，并没有保存到宿主机的文件目录中
