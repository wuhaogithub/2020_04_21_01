部署增强

安装Nginx软件

方法一：apt-get安装

sudoapt-get install -y build-essential libssl-dev libtool libpcre3 libpcre3-dev make openssl zlib1g-dev

sudo apt-get install nginx -y

安装如果出现如下问题：

执行：

sudo rm /var/cache/apt/archives/lock
sudo rm /var/lib/dpkg/lock

方法二：编译安装

省略（参考文档）

检查效果：

ps aux | grep nginx

或者

netstat -tnulp | grep 80

浏览器查看

http://127.0.0.1/

Nginx开启（停止）命令

systemctl start|stop|reload|... nginx

systemctl的本质是 /etc/init.d/nginx start|stop|reload|...

/etc/init.d/nginx 开启的是/usr/sbin/nginx中的nginx程序

我们也可以通过/usr/sbin/nginx ...开启nginx

开启和关闭指令不要混用

nginx启动相关命令

指令 说明
nginx -V 配置参数
nginx -v 版本信息
nginx -s stop 停止nginx
nginx -s reload 重启nginx
nginx -t 检查默认配置文件
nginx -t -c file.conf 检查指定配置文件

移除相关命令：

查看和nginx相关软件

dpkg --get-selections|grep nginx

移除nginx，包括相关文件

sudoapt-get --purge remove nginx

sudo apt-get --purge remove nginx-common

sudo apt-get --purge remove nginx-core
